# README

## Name: Robot World
 
## Overview: 
自分が製造したロボットを販売できるサイト。
ユーザーはスターで５段階評価ができ、コメントも登録できる。
deviseを使用し、ユーザー登録もできる。


## Function:
使用したGems:
```
1. devise
2. bootstrap-sass
3. carrierwave(mini_magick)
4. jquery-rails
5. fog
6. faker
7. aws-sdk
```

Description
Rails: 5.1.6
